    ② Live board for other viewers

    Same-screen tabs/windows update instantly on their own. To let people on other devices watch, publish a snapshot and commit it to the repo — their boards auto-refresh from it.

    How to publish (one time setup, then one commit per update)
    1. Click Publish snapshot — it downloads board-state.json.
    2. Drop that file into the repo root (replace the old one) and commit/push: git add board-state.json && git commit -m "update board" && git push
    3. GitHub Pages redeploys in ~1 min; every open board polls and updates itself.
